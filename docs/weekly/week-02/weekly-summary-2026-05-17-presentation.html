Food Input04 / 13
Food Recognition

식품 인식 파이프라인

Input

사용자 촬영 이미지

접시, 포장, 혼합 음식 사진을 입력으로 받아 음식 영역과 후보 class를 분리합니다.

Output

다중 후보와 신뢰도

결과는 확정 기록이 아니라 화면 검토용 후보입니다. 낮은 신뢰도와 class 충돌은 review 대상으로 남깁니다.

이번 주 정리: 비전 모델 응답을 음식 후보, 위치 정보, 신뢰도, review flag로 정규화하는 흐름을 확인했습니다.
기술 축: YOLO 계열 모델, Google Cloud Vision 후보, 초기 20개 음식 class

Supplement OCR

Input

영양성분표, 원재료명, 섭취 방법 등 분석 가치가 높은 영역을 분리해 OCR 품질을 관리합니다.

Supplement Review08 / 13
Analysis Preview

승인 전 분석 계층

성분 후보, 함량 후보, 주의 후보를 기록 저장과 분리된 preview 상태로 관리합니다.

성분명과 함량은 editable candidate로 노출해 OCR 오인식과 단위 오류를 교정할 수 있게 합니다.
주의 후보는 확정 판단이 아니라, 사용자 검토와 전문가 상담 안내가 필요한 정보로 제한합니다.
승인 전 preview는 복용 기록, 알림, Agent context에 side effect를 만들지 않습니다.
