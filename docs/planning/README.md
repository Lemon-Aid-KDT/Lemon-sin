# Planning

서비스 기획, 제품 기준, 도메인 학습 자료, 연구 근거, 부록성 실행 자료를 둡니다.

## 문서 목록

| 폴더 | 내용 |
|------|------|
| [guide/](./guide/) | 제품·개발 기준 가이드 |
| [domain/](./domain/) | 만성질환, 영양, 식단 등 도메인 학습 자료 |
| [research/](./research/) | 논문, API, 외부 자료, 기술 판단 근거 |
| [appendices/](./appendices/) | 본문 가이드에서 분리한 부록성 실행 자료 |

## 작성 기준

- 계속 유지되는 기준 문서는 `guide/`에 둡니다.
- 외부 근거와 기술 판단 자료는 `research/`에 둡니다.
- 발표용 압축본은 `presentations/` 또는 `weekly/`에 둡니다.
