# GLOBAL_REVIEW Brief

> 확인일: 2026-05-14  
> 읽은 자료: AI 식품·영양 섭취 측정 scoping review 공개 원문/초록, 모바일 음식 인식 리뷰 공개 요약, LLM 의료 안전성 논문 공개 본문 일부, PubMed 공개 메타데이터

## 핵심 요약

AI 기반 식품·영양 섭취 측정 scoping review는 2010~2023년 연구 25건을 검토했다. 전통적인 24시간 회상, 식사기록, 식품섭취빈도조사는 기억 오류와 과소·과대 보고가 생기기 쉽고, AI는 이미지·소리·턱 움직임·텍스트를 이용해 식품 섭취 측정을 보완할 수 있다고 설명한다.

AI 식단 평가 과정은 보통 음식 이미지 여부 확인, 음식 종류 인식, 음식 분할, 제공량·잔량 추정, 영양소 추정으로 이어진다. 각 단계의 오류가 누적되며, 복합 음식, 다양한 조리법, 조명, 촬영 각도, 음식 문화 다양성, 영양성분표 오차가 큰 한계다.

모바일 음식 인식·부피·열량 추정 리뷰는 모바일 비전 기반 앱이 아직 신뢰와 설명 가능성 면에서 초기 단계라고 본다. 사용자가 어떤 음식 조각이 어떤 영양정보와 연결되는지 이해할 수 있어야 한다.

환자 질문에 대한 LLM 안전성 연구는 공개 챗봇이 의료 질문에 문제 있는 답변을 줄 수 있음을 보여준다. 특히 병력 청취 누락, 중요한 경고 누락, 사용자 맥락을 고려하지 않은 답변이 위험하다.

## Lemon Aid 반영

- 음식 사진 분석은 확정값이 아니라 후보값으로 표시한다.
- 분량, 조리법, 국물 섭취량, 복합 음식은 사용자 확인을 받는다.
- LLM은 공식 DB 결과를 설명하는 역할로 제한하고, 의료적 판단이 필요한 질문은 상담 권장으로 넘긴다.

## 사용 금지선

국외 논문의 수치나 추천 기준을 한국 사용자에게 직접 적용하지 않는다. LLM이 개인별 식단·복약·질환 판단을 생성하게 하지 않는다.
