### DRI 개념
- EAR (Estimated Average Requirement, 평균필요량) — 50% 인구 충족
- RDA (Recommended Dietary Allowance, 권장섭취량) — 97-98% 인구 충족
- AI (Adequate Intake, 충분섭취량) — RDA 산출 어려울 때
- UL (Tolerable Upper Intake Level, 상한섭취량)

## 📊 참조 우선순위 (법무·임상 검수용)
