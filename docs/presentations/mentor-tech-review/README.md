# Mentor Tech Review Docs

레몬헬스케어 CTO 멘토에게 Lemon Aid의 기술 선택과 확장 가능성을 검토받기 위한 보조 자료입니다.

## 문서 구성

| 문서 | 목적 |
|------|------|
| [technology-decision-report.md](./technology-decision-report.md) | 기술 선택, 근거, CTO 확인 질문 원문 |
| [lemon-aid-mentor-tech-review.html](./lemon-aid-mentor-tech-review.html) | 기존 멘토 발표 자료와 같은 가로형 PDF 생성을 위한 HTML |
| [lemon-aid-mentor-tech-review.pdf](./lemon-aid-mentor-tech-review.pdf) | CTO 멘토 공유용 PDF |

## 작성 기준

- 기존 첫 미팅 PDF는 수정하지 않고, 이 자료는 기술 검토용 별도 문서로 둡니다.
- 새 기술 후보를 추가하지 않고 현재 guide 문서에 있는 기술 선택만 정리합니다.
- CTO에게 확정받아야 할 항목은 질문으로 표현합니다.
- 건강 관련 표현은 진단, 치료, 처방이 아니라 참고 정보와 전문가 상담 권장 기준을 유지합니다.
