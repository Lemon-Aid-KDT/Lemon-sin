## 작성 기준

- 서비스 방향 요약 문서이므로 세부 기술보다 문제, 해결 방향, 기대효과를 먼저 보여줍니다.
- 사용자는 특정 연령대가 아니라 만성질환자와 건강관리가 필요한 사용자로 정의합니다.
- 건강 관련 표현은 진단, 치료, 처방으로 보이지 않게 쓰고, 참고 정보와 전문가 상담 권장 표현을 유지합니다.
- 외부 확인 질문은 별도 자료로 분리하고 이 문서에는 포함하지 않습니다.
