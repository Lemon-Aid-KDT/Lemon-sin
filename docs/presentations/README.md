# Presentations

미팅, 발표, 공유용 PDF/HTML 산출물을 둡니다.
발표별로 하위 폴더를 만들고, 해당 폴더의 `README.md`에 목적과 산출물을 정리합니다.

## 문서 목록

| 폴더 | 목적 |
|------|------|
| [mentor-first-meeting/](./mentor-first-meeting/) | 첫 미팅용 서비스 개념과 MVP 구현 제안 자료 |
| [mentor-tech-review/](./mentor-tech-review/) | 기술 구조, 구현 범위, 의사결정 검토 자료 |

## 작성 기준

- 발표용 문서는 핵심 메시지와 공유 목적을 먼저 둡니다.
- 세부 기술 근거는 `guide/`, `research/`, `reports/` 문서로 연결합니다.
