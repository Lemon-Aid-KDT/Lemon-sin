# Lemon Aid 문서 아카이브

`changmin-plan` 브랜치는 발표와 공유를 위한 문서 산출물만 보관합니다.
브랜치 루트에는 이 `docs/` 폴더만 두며, 포함 대상은 Markdown, PDF, PPT/PPTX,
발표용 HTML입니다.

## 문서 분류

| 폴더 | 내용 |
|------|------|
| [planning/](./planning/) | 기획서, 제품 가이드, 도메인 학습, 연구 근거, 부록 문서 |
| [daily/](./daily/) | 일일 작업 보고, 검토 보고서, 의사결정 기록 |
| [weekly/](./weekly/) | 주차별 공유·발표 정리 문서 |
| [presentations/](./presentations/) | 멘토 미팅, 기술 리뷰, 발표용 PDF/HTML 산출물 |

## 관리 기준

- 생성 스크립트, 에이전트 설정, 테스트 파일은 이 브랜치에 포함하지 않습니다.
- 새 문서를 추가할 때는 목적 기준으로 위 폴더 중 하나에 배치합니다.
- 문서 위치를 바꾸면 이 인덱스와 해당 폴더의 `README.md`를 함께 갱신합니다.
