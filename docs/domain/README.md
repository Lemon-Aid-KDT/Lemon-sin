# Domain Docs

만성질환, 영양, 식단처럼 팀이 제품을 이해하기 위해 읽는 내부 학습 자료를 둡니다.
사용자에게 직접 노출하는 건강 정보 콘텐츠나 의료 조언 문서는 아닙니다.

## 문서 목록

| 문서 | 목적 |
|------|------|
| [domain-onboarding.md](./domain-onboarding.md) | 만성질환자 영양·식단 도메인 기초 학습 |

## 작성 기준

- 진단, 치료, 처방처럼 보이는 표현을 피합니다.
- 외부 근거가 필요한 내용은 [research](../research/) 문서와 연결합니다.
- 사용자 화면에 직접 쓸 문구는 [Compliance & Safety](../guide/08-compliance-safety.md)를 함께 확인합니다.
