# Lemon-Aid App Icon Assets

## Included
- 00_master: 1024x1024 master PNG icons and transparent mascot source.
- iOS_iPadOS/AppIcon.appiconset: Xcode-ready iOS/iPadOS AppIcon set with Contents.json.
- iPadOS_Only: iPadOS convenience PNGs.
- Android/res: Android launcher, round launcher, adaptive icon foreground/background/monochrome resources.
- Android/play_store: Google Play 512x512 icon.
- PWA_Web: web, PWA, maskable, and Apple touch icon PNGs.
- preview_lemonaid_app_icons.png: visual preview sheet.

## Android placement
Copy the subfolders inside Android/res into:
app/src/main/res/

## iOS/iPadOS placement
Copy AppIcon.appiconset into:
Assets.xcassets/

## Design direction
The icon keeps the Lemon-Aid mascot's bright yellow body, green leaf, glossy soft texture, and friendly face.
The composition is centered for small launcher sizes and uses a soft light-yellow/green background for visibility.
