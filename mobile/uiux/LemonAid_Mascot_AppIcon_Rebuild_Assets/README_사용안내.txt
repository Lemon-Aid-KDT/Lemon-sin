Lemon-Aid 캐릭터 모바일 앱 아이콘 에셋 패키지

포함 포즈
- 01_tablet: 태블릿 사용
- 02_hello: 인사
- 03_search: 돋보기 탐색
- 04_resting: 휴식/잠

폴더 구성
1. 00_master_png_transparent
   배경 제거 PNG 원본 1024x1024, 2048x2048

2. 01_iOS_iPadOS
   iPhone/iPadOS 앱 아이콘용 PNG 크기 세트와 Contents.json

3. 02_Android
   Android launcher icon density별 PNG, adaptive icon XML, foreground, round icon, Google Play 512x512

4. 03_Web_PWA
   Web/PWA용 icon, maskable icon, manifest.webmanifest

5. 04_SVG
   SVG 파일. 원본 캐릭터가 래스터 이미지이므로 SVG 내부에는 PNG가 참조됩니다.

6. 05_AI_Illustrator_Compatible
   Illustrator에서 열 수 있는 PDF 기반 .ai 호환 파일.
   순수 벡터 드로잉이 아니라 원본 래스터 캐릭터를 포함한 편집용 파일입니다.

7. 06_layers
   background / shadow / mascot cutout / final icon / rounded preview 분리 PNG
