## 기능 A: 인원 검색 및 조직도

### 현재 구현 완료

- 자연어 검색 + 48개 부서 별칭 매핑
- HTML/CSS 카드형 전체 조직도 (7본부 + 해외법인 6개소)
- 부서별 직급 행 그리드 레이아웃
- Plotly 바/파이 차트 (공장별 인원, 직급별 분포)

### v3.0 고도화 방안

| # | 고도화 내용 | 상세 | 우선순위 | 대상 부서 |
|---|-----------|------|---------|----------|
| A-1 | **검색 결과 CSV/XLSX 내보내기** | "품보팀 인원 목록 엑셀로" → pandas DataFrame → st.download_button | ⭐⭐⭐ | 총무인사팀, 전 부서 |
| A-2 | **조직도 PDF 인쇄 기능** | HTML 조직도 → PDF 변환 (회의 자료용 출력) | ⭐⭐ | 관리본부, 기술교육원 |
| A-3 | **즐겨찾기/최근 검색 이력** | 자주 찾는 인원 빠른 접근 + 최근 검색 10건 표시 | ⭐⭐ | 전 부서 |
| A-4 | **사원 프로필 상세** | 담당 업무, 보유 자격증, 프로젝트 이력, 교육 이수 현황 | ⭐ | 총무인사팀 |
| A-5 | **조직 변경 이력** | 부서 이동, 직급 변경, 입/퇴사 타임라인 | ⭐ | 총무인사팀 |

### 예상 기술 스택 추가

```
- pandas (DataFrame → CSV/XLSX 변환)
- weasyprint 또는 pdfkit (HTML → PDF 변환)
- st.session_state 기반 검색 이력 캐시
```

## 기능 B: 문서 검색 및 작성

### 현재 구현 완료

- 내부용/외부용 2탭 × 하위 2탭 (양식 다운로드 / 문서 작성)
- 13종 문서 유형 (이메일, 8D, ECN, PPAP, FMEA, 규제 5종, 품질/안전/물류 4종)
- AI 생성 + 직접 편집 + AI 수정 + RESET
- DOCX/PDF/HWPX/TXT 내보내기

### v3.0 고도화 방안

| # | 고도화 내용 | 상세 | 우선순위 | 대상 부서 |
|---|-----------|------|---------|----------|
| B-1 | **문서 버전 관리** | 같은 문서의 수정 이력 저장/비교 (diff 표시) | ⭐⭐⭐ | 품질보증팀, 부품개발팀 |
| B-2 | **문서 승인 워크플로우** | 작성 → 검토 → 승인 3단계 + 알림 + 서명 | ⭐⭐⭐ | 품질보증팀, 구매팀 |
| B-3 | **과거 문서 기반 자동 채움** | 이전 8D에서 부품명/품번/고객사 자동 추천 | ⭐⭐ | 품질보증팀 |
| B-4 | **문서 템플릿 즐겨찾기** | 자주 쓰는 양식 상단 고정 + 사용 빈도 통계 | ⭐⭐ | 전 부서 |
| B-5 | **이메일 직접 발송** | 생성된 문서를 SMTP로 바로 전송 (첨부파일 포함) | ⭐ | 영업팀, 구매팀 |
| B-6 | **다국어 문서 동시 생성** | 한국어/영문 버전 병렬 생성 (현재는 수동 전환) | ⭐ | 해외지원팀, 기술영업팀 |

### 예상 기술 스택 추가

```
- SQLite 문서 이력 테이블 (document_versions)
- smtplib / email.mime (이메일 발송)
- difflib (문서 diff 비교)
- 승인 워크플로우 상태머신 (pending → reviewed → approved)
```
