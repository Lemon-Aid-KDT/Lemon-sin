**브랜드 연결**: 아진산업 CI 골드(`#F9A70D`) → 라이트 적응 딥 골드(`#C88A00`)

| 비율 | 역할 | 색상 | 적용 요소 |

### 3-4. 라이트 모드 CSS 아키텍처

**문제**: 다크 CSS(HUD_CSS)가 항상 먼저 주입 → 하드코딩 배경(`#1c2636`)이 라이트 모드에서도 유지
**해결**: 3단계 방어 체계

| 단계 | 방법 | 적용 |
|------|------|------|
| 1단계 | **CSS 변수 기반 전환** | 다크 CSS의 버튼/입력/사이드바 배경을 `var(--hud-surface2)` 등으로 교체 |
| 2단계 | **Specificity 강화** | 라이트 CSS에 `.stApp` 접두어 추가 (0-1-1 → 0-2-1) |
| 3단계 | **최종 보장 규칙** | `html body .stApp .stButton > button` (specificity 0-2-3) |

**변수 전환 매핑**:
```
다크 → 라이트
--hud-surface2: #1c2636 → #F5F1EB
--hud-surface:  #141c28 → #FFFFFF
--hud-bg:       #0c1018 → #FAF8F5
--hud-primary:  #f9a70d → #C88A00
```
