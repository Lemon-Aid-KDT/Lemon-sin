## 0. v3.5 업데이트 요약

### 핵심 변경사항 (9개 항목)
1. **문서 양식 CSV/Excel 다운로드**: 기능 B 문서 작성 + 검색 패널에 XLSX/CSV 내보내기 추가
2. **SPC 공정 데이터 관리**: 통계적 데이터 생성기 + CSV 업로드 인터페이스
3. **AI 도우미 다운로드 버튼 영구화**: 스트리밍 후 rerun에도 다운로드 버튼 유지 (session_state 기반)
4. **인사관리 이력/보안 탭 통합**: 이력 탭을 보안 탭에 통합 (Tier 4: 7→6탭, Tier 3: 5→4탭)
5. **로그인 이력 CSV/Excel 다운로드**: 날짜 필터 + 타임스탬프 포함 내보내기
6. **Light 모드 로고 수정**: 테마별 SVG 전환 (흰색 텍스트 → 다크 브라운)
7. **법규 업데이트 탭 구조 개선**: 서브탭 → 메인 탭 승격, 시나리오 대시보드 통합
8. **법규 업데이트 한글 깨짐 수정**: U+FFFD 19개 복원 + change_detector 안전 잘림 + UTF-8 PRAGMA
9. **크롤링 실행 UI 상단 이동**: 실행 버튼을 테이블 위로 이동 + 빠른 현황 요약 라인

### 추가 변경사항
- **Ollama 모델 정리**: 26개 → 11개 (15개 삭제, 68.3GB 절약)
- **모델 프로필 재구성**: 5개 패밀리 (Qwen3.5/EXAONE/Gemma4/GPT-OSS/Nemotron)
- **비전 모델 교체**: qwen3-vl/llama3.2-vision/gemma3 → Gemma 4 (멀티모달)
- **대시보드 갱신**: 6개 모듈 카드 + v3.5 시스템 정보 반영
- **버전 표기**: v3.4 → v3.5 전체 갱신
