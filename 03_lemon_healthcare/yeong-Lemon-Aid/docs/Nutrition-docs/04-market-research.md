## 1. 시장 규모와 성장 동력

### 1.1 핵심 시장 지표

| 시장 | 2024년 규모 | 성장률 | 출처 |
|------|------------|--------|------|
| **한국 건강기능식품** | 약 6조 440억 원 (2024 추정) → 6조 4,000억 원 (2025) | 2025년 +5.2% | 한국건강기능식품협회 |
| **한국 디지털 헬스케어** | 2023년 6.5조 원 (전년 대비 +14%) | 소프트웨어 의료기기 견인 | 한국디지털헬스산업협회 |
| **글로벌 디지털 헬스케어** | 2023년 2,609억 달러 → 2033년 1조 9,209억 달러 전망 | CAGR 22.10% | Spherical Insights |
| **건강기능식품 구매 가구 비율** | 82.1% (가구당 연평균 약 33.9만 원) | — | 한국건강기능식품협회 |
| **건강기능식품 온라인 구매 비중** | 2024년 54% → 2025년 59% | 매년 5%p 상승 | 마크큐비전 |

### 1.2 성장 동력 5가지

| # | 동력 | 본 프로젝트와의 연관성 |
|---|------|---------------------|
| 1 | **고령화 → 만성질환 증가** | 65세 이상 인구 19.2% (2025) → 페르소나 B와 직접 연결 |
| 2 | **포스트 코로나 건강 관심** | 면역·예방 관련 영양제 수요 ↑ → 영양제 분석 기능 |
| 3 | **AI·LLM 기술 보편화** | OCR + LLM 파이프라인이 모바일 앱에 적용 가능한 수준 도달 |
| 4 | **웨어러블 보급** | HealthKit·Health Connect 자동 연동 가능 |
| 5 | **개인화 트렌드** | "맞춤형 건강기능식품" 시장 2025년 약 4,500억 원 추산 (한국보건산업진흥원) |

### 1.3 시장의 그늘 — 정체 신호

- 건강기능식품 시장은 **2022년 6.1조 원 정점 후 2023년 6.1조 원, 2024년 5.9조 원으로 소폭 감소**
- 외형 성장은 멈췄으나 **"성분 검증 → 가격 비교 → 정기 구독"** 으로 소비 패턴은 고도화
- 소비자의 71%가 **식약처 기능성 인정 여부**를 구매 기준으로 삼음 (한국소비자원, 2025)
- → **"단순한 영양제 추천"** 이 아니라 **"근거 기반 분석"** 이 차별화 키워드

## 2. 시장 세분화 — 4분면 분석

```
                    [의료/임상 깊이 ↑]
                            │
                            │
         마이약         🎯 ← 본 프로젝트 (B 페르소나)
       (처방약 중심)        │ 의료기관 연계 + 만성질환
                            │
   ─────────────────────────┼─────────────────────────
   [단일 영역]              │              [통합 영역]
                            │
       식사진             필라이즈
    (한국 음식 특화)    (영양제 + 식단)
                            │
       CalZen             홈스픽
   (음식 사진 칼로리)   (영양제 추천+커머스)
                            │
                    [일반 사용자 / 웰니스 ↓]
```

### 빈자리 위치

- **상단 좌측 (의료 + 단일)**: 마이약이 점유 (처방약 중심)
- **상단 우측 (의료 + 통합)**: 🎯 **본 프로젝트의 진입 영역** — 비어 있음
- **하단 좌측 (웰니스 + 단일)**: 식사진, CalZen
- **하단 우측 (웰니스 + 통합)**: 필라이즈, 홈스픽 — 가장 경쟁 치열

> 💡 **결론**: 일반 사용자(B2C 웰니스) 영역은 이미 포화. **의료/임상 깊이 + 통합 분석**이라는 빈 사분면이 차별화 영역.

## 3. 경쟁사 상세 벤치마킹

### 3.1 필라이즈 (Pillyze) — 카테고리 1위

#### 회사 개요

| 항목 | 내용 |
|------|------|
| **운영사** | Pillyze Inc. (서울 강남구 역삼동) |
| **핵심 인력** | 카테고리 1위 플랫폼 빌딩 경험 보유 연쇄 창업팀 |
| **출시** | 2022~2023년 (정확한 출시일 비공개) |
| **사명 의미** | Pill(영양제) + Analyze(분석) 합성어 |
| **MAU** | 75만 (출시 14개월 만에 달성) |
| **누적 영양제 조합 분석** | 85만 회 |
| **투자 유치** | 120억 원 (Strong Ventures, Capstone Partners 리드) |

#### 핵심 기능

1. **영양제 조합 분석 AI '파나케이아' (특허)**
   - 국내외 영양제 DB 2만 개
   - 부족·과도 성분, 부작용 위험 성분 식별
2. **10년치 건강검진 기록 분석**
   - 21개 검진 항목 자동 불러오기
   - 혈압·혈당·체질량 추이
3. **식단 점수 + 칼로리 계산**
   - 사진 기반 음식 인식
4. **영양제 리뷰·랭킹**
   - 사용자 작성 효과·부작용 후기
5. **AI 코치 '피티쌤'**
   - 건강 코칭 챗봇
6. **영양제 섭취 알림**
7. **HealthKit 연동** (걸음수, 활동 에너지)

#### 신뢰 출처

식품의약품안전처 / 식품안전정보원 / 한국영양학회 / EFSA / 대한약학회 / NIH

#### 비즈니스 모델

- **프리미엄 구독 모델** (Free + Premium)
- 약사·영양사 등 전문가 상담 (유료)
- 광고 없음 강조

#### 강점 (Strengths) ⭐

- ✅ **카테고리 1위** — 75만 MAU, 시장 지배력
- ✅ **DB 규모** — 2만 개 영양제 (검색 등록 가능)
- ✅ **AI 차별화** — '파나케이아' 특허 → 진입 장벽
- ✅ **검진 기록 통합** — 일반 영양제 앱이 못하는 영역
- ✅ **신뢰 출처** — 정부·학회 데이터 인용으로 신뢰도 확보
- ✅ **빠른 사용자 획득** — 14개월 75만 MAU
- ✅ **자본력** — 120억 투자로 마케팅·R&D 여력

#### 약점 (Weaknesses)

- ❌ **사용자 입력 의존** — 검진 기록도 사용자가 등록해야 함
- ❌ **의료기관 직접 연계 부재** — 진료·처방 데이터 없음
- ❌ **만성질환자 특화 기능 미흡** — 일반 사용자 대상
- ❌ **영양제 등록 방식이 검색 중심** — 사진 스캔 자동화 약함
- ❌ **연동 한계** — 사용자 후기에 "샤오미 미밴드, 미피트니스 연동 안 됨" 불만

#### 본 프로젝트 시사점

> **"필라이즈를 정면으로 따라가지 말고, 그들이 못하는 곳으로 가라."**

- ✅ 학습할 점: 직관적 UI, 신뢰 출처 명시, 검진 기록 통합 발상
- ✅ 다른 전략: **의료기관 직접 연계 (LDB)**, **사진 스캔 자동화**, **만성질환자 특화 v4**
- ✅ 시장 검증: 75만 MAU = "AI 영양제 분석" 카테고리에 시장 수요가 분명히 있음

### 3.2 마이약 (MyYak) — 처방약 중심 통합 관리

#### 회사 개요

| 항목 | 내용 |
|------|------|
| **운영사** | (정보 비공개, 한국 스타트업) |
| **출시** | 2023~2024년 추정 |
| **포지셔닝** | "AI 처방약/영양제 알림 + 건강상태 자동 분석" |
| **유저 규모** | 비공개 (Google Play 등록) |

#### 핵심 기능

1. **AI 처방전 스캔** — 사진 한 장으로 처방약 자동 등록
2. **처방약 작용 설명** — 약이 몸에서 어떻게 작용하는지 쉽게 설명
3. **약-음식 상호작용 안내** — 함께 먹으면 안 되는 음식
4. **함께 먹으면 좋은 영양제 추천**
5. **영양제 검색 등록** (스캔 X, 검색 후 등록)
6. **복약 알림** — 시간 맞춤
7. **가족 연동** — 부모님 복약 현황 확인
8. **컨디션 일일 기록** — 약과 컨디션 변화 상관관계 추적
9. **처방전 보관함** — PDF로 보관
10. **카카오 간편 로그인 + 큰 글씨 지원** (어르신 친화)

#### 데이터 출처

- 의약품 정보: 식품의약품안전처 의약품안전나라

#### 비즈니스 모델

- 무료 앱 (광고 없음 강조)
- 유료 모델은 명확히 공개되지 않음

#### 강점 (Strengths) ⭐

- ✅ **처방약 자동 스캔** — 마찰 매우 낮음
- ✅ **약-영양제 상호작용 검토** — 본 프로젝트 페르소나 B의 핵심 니즈
- ✅ **가족 연동** — 부모 케어 자녀 세대 타겟
- ✅ **고령자 친화** — 큰 글씨, 카카오 로그인
- ✅ **식약처 공식 의약품 DB 활용** → 신뢰도

#### 약점 (Weaknesses)

- ❌ **영양제 자동 등록 부재** — 검색 방식만 지원
- ❌ **체중 예측·운동 권고 부재** — 단편적 기능
- ❌ **활동 데이터 연동 약함**
- ❌ **시장 인지도 낮음**

#### 본 프로젝트 시사점

> **마이약은 페르소나 B의 핵심 니즈(약-영양제 상호작용)를 정확히 짚었으나 단일 영역에 머물러 있다.**

- ✅ 학습할 점: 처방약 + 영양제 통합 발상, 가족 연동, 고령자 친화 UX
- ✅ 본 프로젝트 강점: **마이약의 처방약 + 필라이즈의 영양제 분석 + 우리의 운동·체중·식단** = 진정한 통합
- ✅ 직접 경쟁 가능성 낮음 (마이약은 처방약, 우리는 영양·식단·운동 중심)

### 3.3 홈스픽 (Homespick) — 영양제 추천·구매

#### 회사 개요

| 항목 | 내용 |
|------|------|
| **운영사** | Innomax Global |
| **포지셔닝** | "생활을 MRI하다, 모바일 헬스케어" |
| **차별점** | 추천 → 구매 일원화 (e-커머스 결합) |

#### 핵심 기능

1. **비대면 건강 데이터 분석** — 식이·영양·생활패턴
2. **건강기능식품 섭취 알림** — 잔여수량 관리
3. **만보계 + 리워드** — 목표 달성 시 보상
4. **건강기능식품 추천 + 구매** — 전문 MD 집단 큐레이션

#### 비즈니스 모델

- **e-커머스 + 추천** — 건강기능식품 판매 수수료
- 리워드 기반 사용자 락인 (lock-in)

#### 강점 (Strengths)

- ✅ **추천 → 구매 일원화** — 수익 모델 명확
- ✅ **리워드 시스템** — 사용자 참여 유도
- ✅ **MD 큐레이션** — 전문가 추천 신뢰도

#### 약점 (Weaknesses)

- ❌ **AI 분석 깊이 부족** — 필라이즈 대비 약함
- ❌ **분석 < 판매 색채** — "분석 앱"이 아닌 "쇼핑 앱"으로 인식되기 쉬움
- ❌ **만성질환자 특화 부재**
- ❌ **사용자 후기 부재** — 진정성 있는 분석 신뢰 어려움

#### 본 프로젝트 시사점

> **홈스픽의 추천 → 구매 모델은 본 프로젝트의 BM 후보지만, 분석의 진정성을 약화시킬 수 있는 양날의 검.**

- ⚠️ 본 프로젝트는 **분석 신뢰도를 핵심 가치**로 두는 페르소나 B 타겟이므로, e-커머스 통합은 신중하게
- ✅ 향후 확장 옵션으로 고려 (Year 2~3에 추천 알고리즘 + 제휴 커머스)

### 3.4 CalZen — 식단 분석 글로벌 강자

#### 회사 개요

| 항목 | 내용 |
|------|------|
| **운영사** | CalZen (글로벌, 한국어 지원) |
| **포지셔닝** | "사진 한 장으로 끝나는 칼로리 계산" |
| **사용자 규모** | 300만+ |
| **앱스토어 평점** | 4.8★ |

#### 핵심 기능

1. **사진 → 칼로리 3초 분석** — AI가 재료 인식, 양 분석
2. **무제한 사진 기록** (무료 티어)
3. **체중 추이 + 활동 자동 동기화** — 걸음 수, 소모 칼로리, 수분
4. **목표 설정 + 진행 시각화**
5. **체중 추이 우선 표시** UI 철학

#### 비즈니스 모델

- **3일 무료 체험 + 프리미엄 구독**
- 프리미엄 기능: 무제한 스캔, 상세 영양소, 체중 추이, 건강 점수

#### 강점 (Strengths) ⭐

- ✅ **속도 최적화** — MFP(MyFitnessPal) 대비 검색·입력 0초
- ✅ **사용자 마찰 최소화** — 다이어트 앱 시장 1위급 사용성
- ✅ **300만+ 검증된 사용자**
- ✅ **헬스 데이터 자동 동기화** — HealthKit 등

#### 약점 (Weaknesses)

- ❌ **영양제 분석 부재**
- ❌ **만성질환자 특화 부재**
- ❌ **한국 음식 정확도** — 글로벌 모델이라 한국 음식엔 식사진보다 약함
- ❌ **의료적 신뢰도 부재** — 다이어트 도구 색채 강함

#### 본 프로젝트 시사점

> **"사진 → 결과 3초"의 사용성 표준은 따라잡아야 한다. 속도 = 사용자 유지율.**

- ✅ 학습할 점: 입력 마찰 최소화, 체중 추이 우선 UI, HealthKit 자동 연동
- ✅ 직접 경쟁 가능성: 식단 분석 영역에서만 부분 경쟁
- ✅ 본 프로젝트 차별화: **한국 음식 + 영양제 + 의료** 영역

## 6. SWOT 분석 (본 프로젝트 관점)

### 🟢 Strengths (강점)

| 강점 | 근거 |
|------|------|
| **레몬헬스케어의 LDB 의료데이터 인프라** | 130여 의료기관, 180여 표준 API |
| **770만+ 검증된 사용자 베이스** | 청구의신 사용자 풀, 즉시 마케팅 가능 |
| **회사 가이드의 임상적 정교함** | v1~v4 만성질환 가중, KDRIs, Mifflin-St Jeor |
| **공식 데이터 활용** | 식약처·한국영양학회·KDRIs |
| **상장 회사의 신뢰도** | 코스닥 상장으로 의료적 신뢰 확보 |

### 🟡 Weaknesses (약점)

| 약점 | 완화 방안 |
|------|----------|
| **시장 후발주자** | 필라이즈가 이미 75만 MAU 선점 → 만성질환 영역 차별화 |
| **데이터 신선도** | KDRIs는 2020년 기준 — 2025 KDRIs 갱신 시 즉시 반영 필요 |
| **학생 팀 구현 한계** | 10주 일정, 인력 한계 → MVP 범위 명확화로 대응 |
| **DTx 미허가 리스크** | 표현 가이드 엄격화로 회피 |
| **OCR 정확도 미검증** | 100장 테스트셋으로 사전 검증 |

### 🔵 Opportunities (기회)

| 기회 | 활용 방안 |
|------|----------|
| **고령화·만성질환 증가** | 페르소나 B 타겟층 자연 확대 |
| **마이데이터 정책** | 보건복지부 '건강정보 고속도로' 사업 → LDB 연계 가능 |
| **건기식 시장 성숙** | 단순 추천이 아닌 "근거 기반 분석" 수요 ↑ |
| **AI·LLM 보편화** | 영양제 OCR + 파싱 비용 ↓, 정확도 ↑ |
| **레몬헬스케어 IPO 자금** | B2C R&D 자원 확보 → 정식 출시 가능성 |

### 🔴 Threats (위협)

| 위협 | 대응 방안 |
|------|----------|
| **필라이즈의 만성질환 영역 진입** | LDB 직접 연계로 경쟁자가 따라올 수 없는 해자 구축 |
| **글로벌 빅테크 진출** (애플·아마존) | 한국 의료 인프라·법규 진입장벽이 보호막 |
| **의료법·약사법 강화** | 면책 고지 + 법무 검수 + 의료자문위 |
| **개인정보 유출 리스크** | ISMS-P, AES-256, 별도 동의 UI |
| **건기식 시장 정체** | 차별화로 시장 점유율 가져오는 전략 (시장 확대보다 점유) |

## 8. 차별화 포지셔닝 맵

### 8.1 두 축으로 본 포지셔닝

```
                         [의료적 신뢰도 ↑]
                                │
                                │
                  마이약   🎯본 프로젝트
                                │
   ────────────────────────────┼────────────────────────────
   [단편 분석]                  │                  [통합 분석]
                                │
                  CalZen        │       필라이즈
                  식사진        │       홈스픽
                                │
                                │
                         [의료적 신뢰도 ↓ / 웰니스]
```

- **🎯 본 프로젝트**: 의료적 신뢰도 ↑ + 통합 분석 → **빈 사분면**
- 필라이즈: 통합 분석에는 강하나 의료기관 연계 부재
- 마이약: 의료적 신뢰는 높으나 처방약 단편 분석
- CalZen / 식사진: 식단 단편

### 8.2 한 문장 포지셔닝

> *"필라이즈는 영양제만, 마이약은 처방약만, 식사진은 음식만 봅니다.*
> *건강의신은 **영양제·식단·운동·체중·약물 상호작용**을 한 번에 보고,*
> *향후 **의료기관 진료 데이터까지 통합**할 수 있는 유일한 만성질환자 헬스케어 플랫폼입니다."*
