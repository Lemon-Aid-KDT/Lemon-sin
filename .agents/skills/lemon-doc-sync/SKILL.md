# Lemon 문서 동기화

코드, API, schema, 알고리즘, 런타임 Agent, Tool, 화면, 환경변수, 데이터 출처,
컴플라이언스 문구가 바뀔 때 이 skill을 사용합니다.

## 워크플로

1. 변경된 제품 표면을 식별합니다.
2. `docs/guide/09-team-workflow.md`의 17.10 섹션에서 함께 바뀌어야 할 문서를
   확인합니다.
3. 필요한 최소 문서를 갱신합니다.
4. `guide.html`은 절대 직접 수정하지 않습니다.
5. 전체 체크아웃에서는 `python scripts/sync_guide.py --check`를 실행합니다.

## 주요 파급 규칙

- 새 API: API 표, 데이터 모델, 호출 흐름, 파일 구조, 담당자 정보를 갱신합니다.
- 새 런타임 Agent 또는 Tool: Agent 표, Tool 표, 호출 흐름, 파일 구조를 갱신합니다.
- 새 화면: 제품 명세, 파일 구조, 팀 담당, UI 문서를 갱신합니다.
- 새 환경변수: `.env.example`, 시크릿 문서, 배포 환경변수 표를 갱신합니다.
- 새 의료 표현 케이스: 안전 가이드를 갱신하고 테스트를 추가하거나 수정합니다.
