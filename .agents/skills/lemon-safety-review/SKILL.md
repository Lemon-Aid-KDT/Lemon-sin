# Lemon 안전 리뷰

건강 로직, 영양/영양제 문구, 복약 인접 동작, 사용자 동의, 개인정보,
프롬프트, Tool, 런타임 Agent 변경에 이 skill을 사용합니다.

## 리뷰 체크리스트

- 진단, 치료, 처방, 효과 보장 표현이 없습니다.
- 특정 의약품이나 영양제가 누구에게나 안전한 것처럼 제시되지 않습니다.
- 입력, OCR, 모델 추출이 불완전할 수 있는 경우 불확실성이 드러납니다.
- 사용자 노출 권고는 정보 제공에 머무르고, 필요한 경우 전문가 상담을 권장합니다.
- 개인 건강 데이터와 시크릿이 로그, 출력, 커밋에 노출되지 않습니다.
- AI가 생성한 액션은 저장 전에 사용자 미리보기와 승인을 거칩니다.

## 출력

구체적인 findings를 먼저 제시합니다. 파일 경로와 가장 작은 안전한 문구 또는
동작 변경안을 포함합니다. 문제가 없으면 남은 위험과 확인한 범위를 명시합니다.
